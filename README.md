# Dotfiles & Config Files

Some dot and config files I use for my system

Symbolic links:  
```
ln -s .zshrc ~/.zshrc
ln -s /home/fredrik/dotfiles/.tmux.conf /home/fredrik/.tmux.conf
ln -s ./foot/foot.ini ~/.config/foot/foot.ini

ln -s ./autostart/start-pipewire.sh ~/config/autostart/start-pipewire.sh

ln -s ./screenshot.sh ~/.local/bin/screenshot.sh
ln -s ./bin/start_dwl.sh ~/.local/bin/start_dwl.sh
```
